Bing 每日一图 API

提供获取必应每日一图的 RESTful API 接口，支持指定日期、随机图片和多种分辨率。

URL
https://api.bphoto.xuanbo.top

API 地址

https://api.bphoto.xuanbo.top

参数用法和响应列表

描述 获取今日的 Bing 每日一图，自动使用当前日期
必需 无需参数
示例 URL：
https://api.bphoto.xuanbo.top/
HTTP 响应：
302 成功，重定向到今日图片 URL
HTTP/1.1 302 Found Location: https://www.bing.com/th?id=OHR.TodaysImage_ZH-CN1234567890_1920x1080.jpg
500 服务器错误（JSON 数据获取失败）
HTTP/1.1 500 Internal Server Error Content-Type: application/json { "error": "服务器错误", "message": "无法获取照片数据" }

描述 获取指定日期的 Bing 每日一图
格式 YYYYMMDD（8位数字）
必需 是（除非使用 random 参数或无参数访问）
示例 URL：
https://api.bphoto.xuanbo.top/?date=20251215 https://api.bphoto.xuanbo.top/?date=20241216
HTTP 响应：
302 成功，重定向到指定日期的图片 URL
HTTP/1.1 302 Found Location: https://www.bing.com/th?id=OHR.ChristmasGnomes_ZH-CN4405839101_1920x1080.jpg
400 日期格式错误
HTTP/1.1 400 Bad Request Content-Type: application/json { "error": "时间格式错误", "message": "日期格式应为YYYYMMDD，例如：20251215", "correct_format": "YYYYMMDD", "example": "20251215" }
404 未找到对应日期的图片
HTTP/1.1 404 Not Found Content-Type: application/json { "error": "未找到对应日期的照片", "date": "20200101", "message": "请检查日期是否正确，或该日期暂无照片" }

描述 随机获取一张历史图片
格式 任意值（建议使用 1 或空值）
必需 是（除非使用 date 参数或无参数访问）
示例 URL：
https://api.bphoto.xuanbo.top/?random https://api.bphoto.xuanbo.top/?random=1
HTTP 响应：
302 成功，重定向到随机图片 URL
HTTP/1.1 302 Found Location: https://www.bing.com/th?id=OHR.WinterForest_ZH-CN1234567890_1920x1080.jpg

描述 指定图片分辨率
格式 宽x高（如 1920x1080）
必需 否
可选值：
分辨率 适用设备
1920x1200宽屏桌面
1366x768笔记本电脑
1280x768宽屏显示器
1024x768平板横屏
800x600小屏幕
800x480移动设备横屏
768x1280平板竖屏
720x1280手机竖屏
示例 URL：
https://api.bphoto.xuanbo.top/?size=800x480 https://api.bphoto.xuanbo.top/?date=20251215&size=800x480 https://api.bphoto.xuanbo.top/?random&size=1366x768
HTTP 响应：
302 成功，重定向到指定分辨率的图片 URL
HTTP/1.1 302 Found Location: https://www.bing.com/th?id=OHR.ChristmasGnomes_ZH-CN4405839101_800x480.jpg

错误处理示例

请求：
https://api.bphoto.xuanbo.top/?date=2025-12-15
响应：
HTTP/1.1 400 Bad Request Content-Type: application/json Content-Length: 125 { "error": "时间格式错误", "message": "日期格式应为YYYYMMDD，例如：20251215", "correct_format": "YYYYMMDD", "example": "20251215" }

请求：
https://api.bphoto.xuanbo.top/?date=20200101
响应：
HTTP/1.1 404 Not Found Content-Type: application/json Content-Length: 108 { "error": "未找到对应日期的照片", "date": "20200101", "message": "请检查日期是否正确，或该日期暂无照片" }

当 JSON 数据源不可用时：

请求：
https://api.bphoto.xuanbo.top/
响应：
HTTP/1.1 500 Internal Server Error Content-Type: application/json Content-Length: 72 { "error": "服务器错误", "message": "无法获取照片数据" }

状态码速查表

状态码 含义 典型场景
302 成功重定向 找到图片，重定向到 Bing 图片 URL
400 请求参数错误 日期格式不正确
404 未找到资源 指定日期无对应图片
500 服务器内部错误 JSON 数据源不可用
