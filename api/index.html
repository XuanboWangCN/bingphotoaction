Bing 每日一图 API

基于 RESTful 架构的高性能图片接口，提供必应每日壁纸的获取、历史回溯及多分辨率支持。

HTTPS GET JSON/Image
图册首页
接口基准地址 (Base URL)
https://api.bphoto.xuanbo.top
参数说明与请求示例

不携带任何参数时，默认重定向至必应今日发布的最新壁纸。

请求示例
https://api.bphoto.xuanbo.top/

参数格式： YYYYMMDD (8位数字)
说明： 获取指定日期的历史壁纸。如果日期格式错误返回 400，无数据返回 404。
请求示例 (2025年12月15日)
https://api.bphoto.xuanbo.top/?date=20251215

从历史图库中随机重定向一张壁纸。推荐用于博客背景或随机展示。

https://api.bphoto.xuanbo.top/?random=1

可与其他参数（如 date 或 random）组合使用。默认为 1920x1080。
